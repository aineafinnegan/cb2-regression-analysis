# CB2 Regression Analysis

University coursework project in R focused on regression modelling and survival analysis.

## Overview
This project analyses patient data using a range of regression and survival analysis methods to answer questions about age prediction, treatment type, and survival outcomes.

## Contents
- Assignment report in PDF format

## Methods used
- Simple and multiple linear regression
- Logistic regression
- Kaplan–Meier survival analysis
- Cox proportional hazards modelling
- ROC analysis
- Cook’s distance and model diagnostics
- Proportional hazards testing

## Topics covered
- Predicting patient age from clinical variables
- Predicting treatment type from patient characteristics
- Comparing survival across treatment groups
- Identifying significant covariates affecting survival

## Key findings
- Average meal calorie intake and cognition score were retained in the final age-prediction model
- Meal calorie intake and survival status were significant predictors of treatment type
- Survival differed significantly by treatment group
- Treatment type and cognition score were significant predictors in the final Cox survival model
